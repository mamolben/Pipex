Descripción de las funciones:

count_substr: Cuenta cuántas subcadenas hay en str que están separadas por el carácter delimitador c. Esto es útil para saber cuántas cadenas necesitas para el array resultante.

Lógica: Recorres la cadena y cuando encuentras un carácter distinto de c, comienzas a contar una nueva subcadena. Cuando encuentras el carácter c, simplemente lo ignoras.

copy_substr: Crea una copia de una subcadena que comienza en start y termina antes del siguiente delimitador c.

Lógica: Recorres la subcadena desde start hasta que encuentras el delimitador c o el fin de la cadena.

free_split: Libera la memoria de todas las subcadenas almacenadas en el array array hasta el índice idx.

Lógica: Recorres y liberas cada subcadena individualmente, y luego liberas el array de punteros en sí.

fill_array: Llena el array array con las subcadenas obtenidas de str utilizando el delimitador c.

Lógica: Iteras a través de la cadena str, y para cada subcadena encontrada, la copias con copy_substr y la almacenas en array.

ft_split: La función principal que usa las funciones anteriores para dividir una cadena en subcadenas separadas por el carácter c.

Lógica: Llama a count_substr para obtener el número de subcadenas, luego usa fill_array para llenar el array de subcadenas. Si alguna parte falla, libera la memoria y devuelve NULL.
